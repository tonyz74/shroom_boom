# Shroom Boom
A game for MYP personal project. A gardener tries to exterminate infected plants using their trusty machete, gardening hose, and lawnmower.

# Features
- Flexible gameplay: fight according to your own preference
- 12 levels, roughly 15 minutes of solid gameplay
- Upgrading & leveling + a shop system
- 4 different types of enemies
- A difficult bossfight
- Original pixel art (mostly)
- Score system for benefit to replayability
